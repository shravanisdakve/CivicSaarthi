# Final Regression Agent — CivicSaarthi

## Goal
Verify the app is stable, polished, and competition-ready.

## Strict Rules
- Do not add features.
- Do not refactor unless there is a clear bug.
- Only fix issues that could affect judging, demo, deployment, or user experience.

## Check Areas
1. App builds successfully.
2. Tests pass.
3. No obvious console errors.
4. Login works.
5. Assistant works.
6. Quiz works.
7. Glossary modal works.
8. Language switching works.
9. Map helper works.
10. Navbar/dropdown works.
11. Privacy page works.
12. Source labels and translation prompts work.
13. Mobile layout works.
14. No broken internal links.
15. No misleading claims.

## Commands
Run:
npm test
npm run build
npm run lint

If lint script does not exist or fails due to config only, explain clearly.

## Final Report
Provide:
- final readiness score
- remaining risks
- demo-breaking risks
- exact fixes if needed
- final recommendation: ship or fix
